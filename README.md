# UNAM-I3691CP-PixelPebble-OreGuide
The OreGuide mobile Application is designed to assist students, geologists, engineers, and other users in identifying Namibian ores using a mobile device. Users will be able to capture or upload an image of an ore sample or search by name, colour, or chemical elements.
